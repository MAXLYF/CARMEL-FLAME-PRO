🔥 Firebase Setup

Connect your ESP32 to Firebase Realtime Database. Follow these steps:

1
Go to console.firebase.google.com and create a new project
2
In your project: Build → Realtime Database → Create database (choose "Test mode")
3
Go to Project Settings → Your apps → Add app (Web), register app, copy the config
4
Enable Authentication → Email/Password and create a user account
5
Open this HTML file, find firebaseConfig at the top and replace the values
